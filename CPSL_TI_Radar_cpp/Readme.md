# CPSL_TI_RADAR C++ code

## Installation instructions

## Install pre-requisite pakages

For help with building using cmake, use the following [tutorial](https://cmake.org/cmake/help/latest/guide/tutorial/index.html)

Build the project
```
cd build
cmake ../
cmake --build .
```

Run the project
```
./CPSL_TI_Radar_CPP
```
